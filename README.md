# 🏥 Healthcare Disease Risk Analysis (Data Analytics Project)

## 📌 Project Overview

This project focuses on analyzing healthcare data to identify disease patterns, risk factors, and lifestyle impacts using **Data Analytics techniques **.

The goal is to extract meaningful insights that can help in better healthcare decision-making.

---

## 🎯 Objectives

* Analyze patient health data
* Identify key disease risk factors
* Study impact of lifestyle habits
* Build an interactive dashboard
* Generate actionable insights

---

## 📊 Dataset

* Source: Healthcare dataset (CSV format)
* Records: ~1000 patients
* Features:

  * Age, Gender
  * BMI, Blood Pressure
  * Smoking, Alcohol Consumption
  * Exercise, Family History
  * Diseases (Heart Disease, Diabetes, Stroke, Cancer)

---

## 🛠️ Tools & Technologies Used

* **Excel** (Data storage & initial review)
* **Python**

  * Pandas
  * NumPy
* **Data Visualization**

  * Matplotlib
  * Seaborn
* **Power BI** (Dashboard)
* **GitHub** (Version Control)

---

## ⚙️ Project Workflow

### 1️⃣ Data Loading

* Imported CSV dataset using Pandas

### 2️⃣ Data Inspection

* Checked data types, structure, and summary statistics

### 3️⃣ Data Cleaning

* Removed duplicates
* Checked missing values (minimal cleaning required)

### 4️⃣ Data Transformation

* Converted categorical data into numerical format
* Created new features like Age Groups

### 5️⃣ Exploratory Data Analysis (EDA)

* Disease distribution analysis
* Correlation analysis
* Lifestyle vs disease relationship

### 6️⃣ Data Visualization

* Histograms (Age Distribution)
* Count plots (Lifestyle vs Disease)
* Box plots (BMI vs Disease)
* Heatmap (Correlation)

### 7️⃣ Insights & Findings

* High BMI increases heart disease risk
* Smoking & alcohol strongly impact diseases
* Older age group has higher risk
* Exercise reduces disease probability
* Family history is a major factor

### 8️⃣ Dashboard (Power BI)

* KPI Cards (Total Patients, Disease %)
* Interactive filters (Gender, Age Group)
* Charts:

  * Disease distribution
  * Lifestyle impact
  * Age analysis

---

## 📈 Key Insights

✔ Lifestyle plays a critical role in health
✔ Preventive measures can reduce disease risk
✔ Data-driven decisions improve healthcare outcomes
